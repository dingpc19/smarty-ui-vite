# 特性

前端工程化演示项目。

- Monorepo 风格 (pnpm workspace)
- 独立组件库
- 独立脚手架工具
- SSG 风格文档自动化工具
- seed 项目模板

# 目录说明

```bash
│   packages
│   ├── seed # admin空模板
│   ├── docs  # 文档自动化
│   ├── ui # 组件库
│   ├── admin # admin全量
│   ├── cli # 脚手架工具
```
